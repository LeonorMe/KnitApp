# Malha Marketing Site

This folder contains a self-contained static one-page marketing website for **Malha**.

## Files

- `index.html` - page structure and marketing content
- `styles.css` - responsive visual design
- `script.js` - placeholder waitlist interaction
- `assets/` - local SVG brand and illustration assets

## How to open

Open `index.html` directly in a browser.

No build step, package manager, or server is required.
