# Product Vision

## Working Concept

Malha is an Android companion for knitting and crochet. It helps beginner and intermediate makers plan projects, manage materials, follow instructions, track progress, and share validated craft knowledge.

The app is not only a tracker. It is a calm project lifecycle system for handmade work.

## Target Audience

Primary users:

- Beginners who need simple step-by-step help.
- Intermediate makers who manage several projects, yarns, tools, and saved patterns.
- Users who want to finish more and buy materials more intentionally.

Initial validation audience:

- A small target group of about 10 users.
- Focus on observing whether they can create a project, follow steps, track materials, and resume progress without confusion.

## Core User Promise

Malha helps users:

- Keep track of where they are in a project.
- Avoid losing pattern progress.
- Understand materials and costs.
- Finish more projects.
- Use owned yarn before buying more.
- Get AI help when planning or interpreting instructions.

## MVP Shape

The first Android version should have 3-5 main screens:

1. Home / active projects
2. Project execution
3. Patterns
4. Materials / stash
5. Profile / settings

Community and AI can be previewed in the product model, but the app must remain useful without them.

## Sharing And State

The app should eventually support shared state between users:

- Public patterns.
- Public-to-friends projects.
- Shared project variants.
- Community corrections and reviews.
- A pattern or project object that several users can modify through variants.

The safest model is not live simultaneous editing in the MVP. Instead, support pattern variants and user-owned project progress first.

## Monetization

The simple premium idea:

- Free users can access the core app and a limited set of included patterns.
- For around 1 euro per month, users unlock more than the first 3 included patterns and selected advanced features.

Future premium features:

- Advanced AI assistant.
- Pattern adaptation.
- Yarn estimation.
- Image-to-draft-pattern experiments.
- Premium pattern bundles.

## Positioning

Malha is a trusted knitting and crochet ecosystem with AI assistance.

It should not be positioned as a fully automatic pattern generator from photos, because current AI cannot guarantee exact stitch counts, gauge, shaping, or sizing from images alone.
