# Gamification Economy

## Core Philosophy

Reward creation, not consumption.

The economy should help users feel organized, motivated, and proud. It must not create pressure, shame, or addictive urgency.

## Main Loop

Current common hobby loop:

```text
Buy yarn -> excitement -> start project -> pause project -> buy more yarn
```

Desired app loop:

```text
Plan project -> use materials -> make progress -> earn rewards -> finish project
```

## Currencies

## Craft Coins

Earned through meaningful activity.

Used for:

- Cosmetic themes.
- Small pattern discounts.
- Event entry.
- Extra AI requests later.
- Progress rewards.

## Pattern Credits

Premium currency purchased with real money or included in subscription.

Used for:

- Premium patterns.
- Pattern bundles.
- Marketplace purchases.
- Advanced AI transformations.

## Reputation

Non-spendable trust score.

Earned through:

- Helpful reviews.
- Verified completions.
- Pattern corrections.
- Community support.

Used for:

- Creator trust.
- Beta access.
- People-approved weighting.

## Earning Craft Coins

Reward:

- Active craft sessions.
- Milestones.
- Finished projects.
- Stash usage.
- Gauge swatches.
- Helpful reviews.
- Uploading completion photos.

Avoid rewarding only raw hours. Use a balanced formula:

```text
Coins = active time + progress + completion + consistency + healthy behavior
```

## Anti-Abuse Rules

- Daily earning caps.
- Idle session detection.
- Progress-linked rewards.
- Review reward rate limits.
- Completion rewards tied to plausible project progress.

## Healthy Behaviors

Reward:

- Using owned yarn.
- Finishing paused projects.
- Planning before buying.
- Staying within budget.
- Completing a gauge swatch.

Do not block users from buying or starting projects because they lack coins. Coins should motivate, not punish.

## Marketplace Strategy

The strongest monetization path is a pattern store:

- Verified patterns.
- Beginner bundles.
- Seasonal collections.
- Creator marketplace later.

Subscription can include:

- More included patterns.
- Advanced assistant usage.
- Higher AI limits.
- Premium analytics.

## AI Shopping Assistant

Future flow:

User: "I want a green winter sweater."

Assistant asks:

- Size.
- Skill level.
- Budget.
- Fiber preference.
- Time available.

Assistant returns:

- Pattern options.
- Needed yarn.
- Needles.
- Estimated cost.
- Estimated hours.
- Potential coin rewards.
