# Brand Identity

## Official Brand Direction

The Android app should use the same brand as the marketing site:

- Product name: **Malha**
- AI assistant / mascot: **Aidi**

Strengths:

- Clear Portuguese craft meaning.
- Calm and grounded.
- Already represented in the site assets.
- Aidi works as an AI assistant name.

## Recommendation

Use **Malha** and **Aidi** everywhere in the Android app, documentation, screenshots, and future store copy.

Keep the names in Android string resources so copy remains maintainable, but do not present alternate names in the product.

## Brand Promise

Malha helps makers keep the thread of every project.

## Tagline Options

- Keep the thread of every project.
- Finish more. Waste less. Make calmly.
- Your cozy place for handmade progress.
- Turn yarn into finished projects.
- Plan, stitch, finish.

## Assistant Role: Aidi

Aidi should be:

- Calm.
- Useful.
- Quiet.
- Non-judgmental.
- Practical.

Aidi should not interrupt constantly. It should appear where it helps:

- Project planning.
- Material suggestions.
- Progress nudges.
- Pattern explanations.
- AI-generated drafts.

Example copy:

- "You already have enough yarn for this project."
- "This project is 72% complete."
- "This pattern needs a gauge check before adapting."
- "I can help turn these notes into steps."

## Mascot Direction

Best mascot concept:

- A small yarn/thread-inspired companion.
- Soft geometry.
- Gender-neutral.
- Cute but not childish.
- Works as an app icon, assistant avatar, stickers, and reward cosmetics.

## Visual Identity

Suggested palette:

- Warm cream.
- Deep sage green.
- Soft clay.
- Muted indigo.
- Dusty lavender.

Style:

- Textile-inspired.
- Rounded but structured.
- Calm surfaces.
- Minimal decoration.
- Clear information hierarchy.

## Naming Implementation Rule

All user-facing brand strings must live in `strings.xml`.

Initial resources can use:

- `app_name`: Malha
- `assistant_name`: Aidi

Future naming changes, if any, should only require updates to resources and branding assets.
